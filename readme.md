
# 🌿 Rota Verde 


Rota Verde é um software educacional desenvolvido especificamente para pessoas com dificuldade no uso de tecnologia, oferecendo conteúdos acessíveis e dinâmicos.

## 🎯 Objetivo

Proporcionar uma experiência educacional inclusiva, com interface intuitiva e conteúdos relevantes para o público.

## 🚀 Funcionalidades

- Interface amigável e de fácil navegação
- Conteúdos educacionais variados
- Acessibilidade para usuários idosos
- Suporte ao aprendizado contínuo

## 💻 Tecnologias

- Frontend: [VueJs]
- Backend: [C#]
- Banco de Dados: [postgreSQL]

## 🛠️ Como Instalar

```bash
git clone https://github.com/jamerson-mt/rotaverde-web.git
cd rotaverde-web
npm install
npm run dev
```

## 📋 Requisitos

- Node.js [14.17.0]
